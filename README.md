# Django Course Enrollment Manager

A simple Django project built while learning Django and backend basics.

## Features

- List courses
- View course details
- User login
- Enroll in a course
- Prevent duplicate enrollments
- Course capacity check
- Admin panel for managing data

## Models

- StudentProfile
- Instructor
- Course
- Enrollment

## Run locally

python manage.py migrate
python manage.py createsuperuser
python manage.py runserver

Then open:
http://127.0.0.1:8000
